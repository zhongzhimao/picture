# 微信图片工具箱

## 项目简介

微信图片工具箱是一款一站式解决微信内图片处理需求的小程序，提供免费和付费功能，满足用户各种图片处理需求。

## 功能模块

### 核心功能（免费）
- **图片压缩**：指定大小压缩，保持清晰度
- **格式转换**：支持JPG/PNG/WEBP格式互转
- **基础裁剪/旋转**：图片裁剪、旋转、翻转等基础编辑
- **九宫格切图**：一键切图，完美适配朋友圈

### 高级功能（付费）
- **批量处理**：同时处理多张图片，提升效率（9.9元/月）
- **去除水印**：智能识别并去除图片水印（0.5元/次）
- **证件照换底色**：一键更换证件照背景色（2元/次）
- **AI修复老照片**：智能修复模糊、褪色老照片（3元/次）

## 技术架构

### 前端技术栈
- **框架**：微信小程序原生框架
- **组件系统**：微信小程序组件
- **样式**：WXSS + CSS3
- **图片处理**：Canvas API

### 核心功能实现
- **图片压缩**：使用Canvas进行图片压缩和质量调整
- **格式转换**：Canvas导出不同格式的图片
- **九宫格切图**：Canvas分块导出
- **会员系统**：本地存储用户状态和积分
- **权限控制**：基于用户身份的功能访问控制

## 使用说明

### 免费功能
1. 选择对应功能页面
2. 上传需要处理的图片
3. 调整参数设置
4. 点击处理按钮
5. 保存或分享结果

### 付费功能
1. 开通VIP会员或充值积分
2. 选择高级功能
3. 确认支付
4. 使用功能
